## MP Smart Analysis System (MVP)

מיני‑מערכת מקומית (ללא שרת) שמקבלת **דוח מצב** כטקסט, מזהה מילות‑מפתח/דפוסים, בוחרת **Template** מתאים, ומייצרת **תשובה מקצועית מוכנה** ב‑Markdown.

### איך מריצים
- פתח/י בדפדפן את `index.html`.

### איפה משנים “חוכמה”
- **חוקים/זיהוי**: `rules.js`
- **טמפלטים**: `templates.js`
- **UI**: `index.html`, `style.css`, `app.js`
