## MP Smart Analysis System (MVP)

כלי **מקומי בדפדפן בלבד**: מדביקים דוח מצב, מקבלים טקסט Markdown מוכן. אין שרת יישום, אין API, אין העלאת נתונים לאינטרנט — הכל רץ אצלך במחשב.

### הרצה בדפדפן

הקוד משתמש ב־**ES modules** (`import`). בחלק מהדפדפנים, פתיחה כפול־קליק על `index.html` (`file://`) **חוסמת** טעינת המודולים.

**מה לעשות:** להריץ **צפייה סטטית מקומית** (למשל **Live Server** ב־VS Code / Cursor, או כל שרת סטטי מקומי) **רק כדי לפתוח את הקבצים** — זה לא הופך את המוצר ל“אפליקציה עם שרת”; אין לוגיקה עסקית בצד שרת.

אופציונלי מהטרמינל (אותה מטרה — צפייה מקומית):

```bash
npm run dev:view
```

### בדיקות (אופציונלי)

דורש Node.js מותקן. בודקות את מנוע החוקים ואת **שלמות התבניות בעברית ובאנגלית** (בלי תבניות חסרות).

```bash
npm test
```

### איפה משנים לוגיקה

- **חוקים / ניקוד**: `rules.js`
- **תבניות פלט**: `templates.js` (`HE` / `EN`)
- **ממשק**: `index.html`, `style.css`, `app.js`
