## MP Smart Analyzer

כלי **מקומי בדפדפן בלבד**: מדביקים דוח מצב (טקסט), המערכת מזהה דפוסים עם **מנוע חוקים דטרמיניסטי**, בוחרת תבנית פלט, ומחזירה Markdown מוכן. אין שרת יישום, אין API חיצוני, אין העלאת נתונים לרשת — הכל רץ אצלך במחשב.

### מה כלול בממשק (גרסה נוכחית)

- עיצוב מערכת כהה (קונסולת ניתוח), פאנל קלט / פלט.
- **נוער / בוגרים** — עומס הסברים ומדריך (נוער) מול מינימליזם (בוגרים).
- **מהיר / עשיר** — פלט מיידי וממשק קל מול אפקטים עדינים יותר.
- **שכבת הבנה** (`intelLayer.js`) — סיכום קצר לפני הפלט (סוג קלט, אותות, ביטחון, כיוון) — **פרשנות על בסיס הכללים**, לא מודל שפה חיצוני.
- בקרת איכות קלט בסיסית (דחיית ג’יבריש / קלט לא רלוונטי) לפני יצירת “דוח מקצועי”.

### הרצה בדפדפן

הקוד משתמש ב־**ES modules** (`import`). פתיחה ישירה של `index.html` כ־`file://` עלולה **לחסום** מודולים בחלק מהדפדפנים.

**מה לעשות:** צפייה סטטית מקומית (למשל **Live Server** ב־Cursor/VS Code, או כל שרת סטטי מקומי) — **רק** כדי לטעון קבצים; אין לוגיקה עסקית בצד שרת.

```bash
npm run dev:view
```

### בדיקות (אופציונלי)

דורש Node.js. בודקות את מנוע החוקים ואת **שלמות התבניות בעברית ובאנגלית**.

```bash
npm test
```

### איפה משנים לוגיקה

| אזור | קבצים |
|------|--------|
| חוקים / ניקוד / איכות קלט | `rules.js` |
| תבניות פלט HE/EN | `templates.js` |
| שכבת “הבנה” טקסטואלית (מעל המנוע) | `intelLayer.js` |
| ממשק, זרימה, אחסון מצבים | `index.html`, `style.css`, `app.js` |
| בדיקות חוקים | `tests/rules.test.js` |
| דוח אתר בעברית (ASCII path ל-Git) | `docs/SITE-REPORT-HE.md` |
| רישיון / זכויות יוצרים | `LICENSE` |
| המלצות הרחבות ל-Cursor/VS Code | `.vscode/extensions.json` |
